What is the Matrix Profile?

The matrix profile is a data structure and associated algorithms that helps solve the dual problem of anomaly detection and motif discovery. It is robust, scalable and largely parameter-free.

Why Use The Matrix Profile?

It's fast

The Matrix Profile can compute a quintillion pairwise comparisons in less than a day.

It's easy to use

The Matrix Profile is parameter-free and works straight out of the box.

It's data agnostic

The Matrix Profile has solved challenges in finance, biology, robotics, healthcare, IoT, geology and many other arenas.
